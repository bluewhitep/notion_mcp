# N.I.L.O. Documentation

N.I.L.O. documentation is organized by language. `Docs/EN` is the source documentation tree; `Docs/ZH` and `Docs/JP` mirror the same structure for localized reading.

## Languages

- [English](EN/README.md)
- [中文](ZH/README.md)
- [日本語](JP/README.md)

## Structure

Each language tree keeps the same high-level audience split:

- `User` - installation, configuration, CLI usage, MCP client setup, and troubleshooting.
- `Developer` - architecture, API notes, packaging, tests, MCP tool contracts, and skill templates.

When updating documentation, update `Docs/EN` first, then synchronize the corresponding `Docs/ZH` and `Docs/JP` entries.
