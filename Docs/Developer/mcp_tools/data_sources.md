# Data Source Tools

- `data_source_retrieve`
  - 说明：读取 Notion data source。
  - 必选参数：`data_source_id`。
- `data_source_query`
  - 说明：查询 Notion data source。
  - 必选参数：`data_source_id`。
  - 可选参数：`payload`。
- `data_source_create`
  - 说明：创建 data source。
  - 必选参数：`payload`。
  - 可选参数：`dry_run`。
- `data_source_update`
  - 说明：更新 data source。
  - 必选参数：`data_source_id`、`payload`。
  - 可选参数：`dry_run`。
