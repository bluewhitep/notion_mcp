# Database Tools

- `database_retrieve`
  - 说明：读取 legacy database。
  - 必选参数：`database_id`。
- `database_query`
  - 说明：查询 legacy database。
  - 必选参数：`database_id`。
  - 可选参数：`payload`。
- `database_create`
  - 说明：创建 legacy database。
  - 必选参数：`payload`。
  - 可选参数：`dry_run`。
- `database_update`
  - 说明：更新 legacy database。
  - 必选参数：`database_id`、`payload`。
  - 可选参数：`dry_run`。
